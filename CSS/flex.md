flex는 2개의 개념으로 나뉜다.

첫 번째는 `Container`, 두 번째는 `items`

| 종류      | 속성                                                                    |
| --------- | ----------------------------------------------------------------------- |
| Container | - display <br> - flex-flow <br> - justify-content <br> - align-items 등 |
| items     | - order <br> - flex <br> - align-self 등                                |
