### 공부방법

기본문법 확인

1. CodeUp 100제
2. 백준 1 - 9단계 ([링크](https://www.acmicpc.net/step))
   1. 1-9단계에서 풀지 않아도 되는 문제
      - for문 "빠른 A + B"
      - while문 "A + B"
      - 기본수학1 "ACM 호텔", "Fly me to the Alpha Centauri"
      - 기본수학2 "골드바흐의 추측", "택시 기하학","터렛"
